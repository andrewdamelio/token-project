# Setup
1. ```git clone https://github.com/andrewdamelio/token-project.git```
2. ```cd token-project```
3. ```npm install```
4. ```cd client```
5. ```yarn install```


# Runing

### Sever
1. ```cd token-project```
2. ```node server.js```
3. API: ```http://localhost:8080/```

### Client
1. ```cd token-project/client```
2. ```yarn dev```
3. Client: ```http://localhost:3000/```

# Testing
1. ```cd token-project```
2. ``` npm run test```
